Robots.txt Look-up Python

Robots.txt files provide a useful guide for devs looking to safely web-scrape. This tool makes finding these files more convenient, before you run the web-scraping bot. If nothing else, it may serve as a learning tool.

Once given a valid url, the site will display a "beautified" version of the url's robot.txt and a "how-to-read" guide. Users can also print or save as pdf.
